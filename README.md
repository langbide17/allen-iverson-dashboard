# Allen Iverson Analytics Dashboard

This project is an interactive sports analytics dashboard exploring the career statistics of NBA legend Allen Iverson.

The dashboard demonstrates key data visualization concepts used in analytics including:

• KPI metric cards  
• Interactive filters  
• Performance trend analysis  
• Scatter plot comparisons  
• Season-based filtering  

The goal of this project was to practice dashboard layout, KPI design, and interactive data visualization before applying similar concepts to professional analytics dashboards.

## Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js

## Dashboard Features

- Career performance KPIs
- Season-by-season scoring trends
- Team performance comparison
- Interactive filtering by team and season
- Visual comparison of scoring vs assists

## Development Notes

AI-assisted coding tools were used during development to help generate portions of the dashboard structure. The project was reviewed, organized, and refined as a learning exercise in dashboard design and data visualization.
